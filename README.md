# Earth Intelligence - Space-Tech Website

A modern, cinematic space-tech themed website built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and React Three Fiber.

## Features

- **3D Rotating Globe** - Interactive Earth visualization using React Three Fiber
- **Smooth Animations** - Framer Motion powered scroll animations and transitions
- **Modern Design** - Dark navy theme with cyan accents and glowing borders
- **Responsive Layout** - Fully responsive design for all screen sizes
- **Premium UI** - Clean, futuristic interface without bulky cards

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- React Three Fiber
- Three.js

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
